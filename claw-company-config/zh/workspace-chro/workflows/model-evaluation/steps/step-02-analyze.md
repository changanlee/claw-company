# 步驟 2：對比分析

**進度：步驟 2 / 共 4 步**

### 2. 候選模型評估

對每個候選模型評估以下維度：

- **性能**：回應品質、推理能力、指令遵循度
- **成本**：每百萬 Token 價格（輸入/輸出）
- **上下文長度**：最大上下文窗口
- **速度**：回應延遲（首 Token、完整回應）
- **可靠性**：API 穩定度、Rate Limit
