# 步驟 2：分析指標

**進度：步驟 2 / 共 3 步**

## 目標

根據掃描數據，分析組織健康的三大指標。

## 執行規則

- 📖 讀完整個步驟檔案再開始行動
- 🚫 不預讀後續步驟
- 🚫 不跳步

## 執行指令

### 1. 負載均衡分析

- 各 Agent 任務量是否均衡
- 是否有 Agent 長期過載或閒置
- Token 消耗是否與任務量成正比
- 建議：是否需要調整職責分配

### 2. 跨部門協調分析

- 哪些 Agent 間的協調最頻繁
- 是否有協調瓶頸（某個 Agent 成為所有協調的中心點）
- 資訊流通是否順暢
- 是否有應該協調但未協調的情況

### 3. 能力缺口分析

- 是否有任務無對應 Agent 處理（職責真空）
- 是否有反覆出現的錯誤模式
- 模型能力是否匹配（對照上次 agent-assessment 結果）

### 4. 趨勢對比

與上週數據對比：

| 指標 | 上週 | 本週 | 變化 | 趨勢 |
|------|------|------|------|------|
| 總任務量 | | | | ↑/↓/→ |
| 平均完成率 | | | | |
| 跨部門協調次數 | | | | |
| 異常事件數 | | | | |
