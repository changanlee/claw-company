# 步驟 2：評估適配度

**進度：步驟 2 / 共 4 步**

## 目標

根據收集的數據，評估各 Agent 的能力適配度、任務完成率和模型匹配度。

## 執行規則

- 📖 讀完整個步驟檔案再開始行動
- 🚫 不預讀後續步驟
- 🚫 不跳步

## 執行指令

### 1. 能力適配度評估

對每個 Agent 評估以下維度（1-5 分）：

- **職責覆蓋度**：AGENTS.md 定義的職責是否都有對應的執行紀錄
- **執行品質**：任務完成的品質與準確度
- **回應效率**：處理任務的速度是否合理
- **自主判斷**：綠燈/黃燈/紅燈分級是否得當

### 2. 任務完成率分析

- 已完成任務 / 總分派任務
- 超時完成的比例
- 需要重做的比例

### 3. 模型匹配度評估

對照各 Agent 當前使用的模型（smart / fast），評估：

- 使用 smart 模型的 Agent 是否確實需要高能力模型
- 使用 fast 模型的 Agent 是否出現能力不足的跡象
- Token 消耗與產出品質的性價比
