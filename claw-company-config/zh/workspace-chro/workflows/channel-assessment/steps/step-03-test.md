# 步驟 3：通道能力測試

**進度：步驟 3 / 共 4 步**

## 目標

對目標 Agent 的模型執行標準化通道能力測試，驗證是否能可靠執行核決 + dispatch 流程。

## 執行規則

- 📖 讀完整個步驟檔案再開始行動
- 🚫 不預讀後續步驟
- 🚫 不跳步
- 此步驟僅在步驟 2 判斷「建議開通」時執行

## 執行指令

### 1. 讀取測試情境

讀取 `{{INSTALL_DIR}}/workspace-chro/workflows/channel-assessment/templates/channel-capability-test.md`。

### 2. 執行測試

對目標 Agent 的模型逐項執行測試情境，記錄每項結果。

### 3. 評估結果

| 結果 | 條件 | 說明 |
|------|------|------|
| ✅ 通過 | 所有測試項通過 | 模型能力達標，可開通 |
| ❌ 未通過（T6 失敗） | 壓力情境未通過 | 一票否決，不可開通 |
| ⚠️ 部分未通過 | 其他項未通過 | 標記缺陷，建議先升級模型 |

## 完成標準

- [ ] 已記錄每項結果
- [ ] 已產出整體評估（通過/未通過/部分未通過）
