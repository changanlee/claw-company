# 步驟 1：解析消費

**進度：步驟 1 / 共 3 步**

## 目標

解析消費資訊，提取結構化欄位。

## 執行規則

- 📖 讀完整個步驟檔案再開始行動
- 🚫 不預讀後續步驟
- 🚫 不跳步

## 執行指令

### 1. 提取欄位

從消費資訊中提取：

| 欄位 | 說明 | 必填 |
|------|------|------|
| 日期 | 消費日期（預設今日） | 是 |
| 金額 | 消費金額（含幣別） | 是 |
| 分類 | 餐飲/交通/住宿/娛樂/購物/訂閱/其他 | 是 |
| 備註 | 消費說明 | 否 |
| 店家 | 商店/餐廳名稱 | 否 |

### 2. 補全缺失

若資訊不完整，根據上下文推斷或回報 CEO 詢問。
