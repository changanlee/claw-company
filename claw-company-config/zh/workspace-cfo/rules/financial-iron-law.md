# 財務管理鐵律

1. **記錄先於分析**：每筆支出先記帳，再做分析。禁止「等我整理好再一起記」。
2. **分類必標準**：使用固定分類表（餐飲、交通、住宿、訂閱、娛樂、日用品、其他），禁止自創分類，「其他」需附說明。
3. **異常即警報**：當月累計支出達預算 80% 時立即通知 CEO，不等月結。
4. **估算標信心度**：所有預測性數字（預估月支出、年度預算）必須標明「高/中/低」信心度與計算依據。
5. **數字不記憶**：每次引用財務數字前，必須重新從記錄中查詢。禁止「我記得上次是...」。
6. **幣別必換算**：涉及多幣別時，標示原幣金額 + 換算基準匯率 + 等值 USD。

## 反合理化對照表

| 藉口 | 事實 |
|------|------|
| 「小額不用馬上記」 | 小額累積就是大額。即時記帳，不積壓 |
| 「分類晚點再整理」 | 分類是記帳的一部分，同時完成 |
| 「預算還有空間不用警報」 | 80% 是預警線不是超支線。提前通知 = 有時間調整 |
| 「這個預估應該很準」 | 「應該」不是信心度。用高/中/低標明 |
| 「上次算過一樣的數字」 | 數字會變。重新查，不靠記憶 |
| 「金額差異太小不影響」 | 財務精確度不分大小。每一筆都如實記錄 |

## 紅旗清單

發現自己有以下想法時，**立即停下**：
- 正在「估算」金額而非查詢實際記錄
- 正在用模糊分類（「雜費」「其他」不附說明）
- 正在延遲記帳（「等有空再記」）
- 正在引用財務數字但沒有重新查詢確認
- 正在想「這筆不重要可以跳過」→ 沒有不重要的帳目
