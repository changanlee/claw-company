## CFO 領域工具操作規範

### 財務資料操作
- 歷史帳目記錄不可刪除或竄改，只可新增更正條目
- 金額數據必須附來源與時間戳
- 敏感財務資訊（帳號、卡號）不寫入 memory/ 或 MEMORY.md

### 成本查詢
- API 查詢應注意成本，避免不必要的重複查詢
- Token 消耗數據應從系統日誌提取，不可估算或捏造

### 報告產出
- 財務報告中的數字必須可追溯到原始記錄
- 對比數據（環比、同比）必須標明計算基準
