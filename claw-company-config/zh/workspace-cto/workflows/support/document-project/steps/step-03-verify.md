# 步驟 3：驗證文件準確性

**進度：步驟 3 / 共 3 步** — 最後一步

## 目標

驗證所有撰寫的文件準確性，確保範例可執行、步驟可重現。

## 執行規則

- 📖 讀完整個步驟檔案再開始行動
- 🚫 不預讀後續步驟
- 🚫 不跳步、不合併步驟

## 執行指令

### 1. 範例驗證

檢查文件中的代碼範例：

- 代碼範例是否語法正確
- 指令是否可執行
- 輸出是否與描述一致

### 2. 步驟驗證

檢查操作步驟：

- 安裝步驟是否完整（無遺漏步驟）
- 設定步驟是否正確
- 前後步驟是否有邏輯連貫

### 3. 內容一致性

- 文件中的技術名詞是否與代碼一致
- API 端點是否與實際代碼匹配
- 版本號是否正確

### 4. 產出摘要

整理文件撰寫摘要回報 CTO：

- 完成的文件清單
- 每份文件的位置
- 驗證結果
- 需要注意的事項

## 完成條件

- ✅ 代碼範例已驗證
- ✅ 操作步驟已驗證
- ✅ 內容一致性已確認
- ✅ 摘要已產出

## 下一步

此為最後一步。將摘要回報 CTO。
