# 步驟 3：整合推送

**進度：步驟 3 / 共 3 步**

## 目標

將預測提醒依推送時機分類，整合到晨間簡報或透過 CEO 即時推送。

## 執行規則

- 📖 讀完整個步驟檔案再開始行動
- 🚫 不預讀後續步驟
- 🚫 不跳步

## 執行指令

### 1. 即時推送

若有影響今日安排的緊急預測，立即透過 `sessions_send` 通知 CEO：

> 「CEO，以下預測提醒需即時轉達董事長：[提醒內容]。」

### 2. 晨間簡報整合

將非緊急預測整理為簡報素材，透過 `sessions_send` 提交 CEO，供晨間簡報使用：

> 「CEO，以下為明日預測提醒，請納入晨間簡報：[提醒清單]。」

### 3. 記錄與追蹤

- 將所有預測記錄到 memory/ 日誌，標記推送狀態
- 記錄預測準確度（事後驗證用）：
  - 預測內容
  - 是否成真
  - 準確度評分
- 更新 MEMORY.md 中的預測準確度統計，持續優化

### 4. 靜默結束

若無任何值得推送的預測，靜默結束流程，僅記錄分析結果。

## 完成標準

- [ ] 緊急預測已即時推送（如有）
- [ ] 非緊急預測已提交晨間簡報（如有）
- [ ] 已記錄所有預測供追蹤
- [ ] 已更新預測準確度統計

## 流程結束
