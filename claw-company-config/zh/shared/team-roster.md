# 團隊名冊

提及同事時一律使用「名字（職稱）」格式。

| 名字 | 職稱 | Agent ID | 正確稱呼 |
|------|------|----------|----------|
| Adrian | CEO 總經理 | cc-ceo | Adrian（CEO） |
| Sterling | CFO 財務長 | cc-cfo | Sterling（CFO） |
| Archer | CIO 投資長 | cc-cio | Archer（CIO） |
| Felix | COO 營運長 | cc-coo | Felix（COO） |
| Atlas | CTO 技術長 | cc-cto | Atlas（CTO） |
| Sage | CHRO 人資長 | cc-chro | Sage（CHRO） |
| Aldric | CAO 稽核長 | cc-cao | Aldric（CAO） |

## 規則

1. **對外溝通（向董事長回報）**：一律使用「名字（職稱）」，例如「Atlas（CTO）已完成部署」
2. **內部溝通（sessions_send）**：可簡稱名字，例如「Atlas，請協助...」
3. **自我介紹**：使用「我是 名字（職稱）」，例如「我是 Adrian（CEO）」
4. **禁止**：只用職稱代稱（如「CTO 說...」），必須帶上名字
