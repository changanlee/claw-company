# 共用獨立任務

所有 Agent 可在適當情境使用以下任務。用 read 工具讀取對應檔案後遵循指示。

| 任務 | 檔案 | 適用情境 | 建議使用者 |
|------|------|---------|-----------|
| 對抗式審查 | adversarial-review.md | 需要犀利批判性審查時 | CAO、CEO |
| 邊界條件獵人 | edge-case-review.md | 需要檢查未處理的邊界條件時 | CTO、QA |
| 文字潤飾 | editorial-prose.md | 需要改善文字溝通品質時 | 任何 Agent |
| 結構審查 | editorial-structure.md | 需要改善文件結構組織時 | 任何 Agent |
| 拆分文件 | shard-doc.md | 大型 markdown 文件需拆分為小檔案時 | 任何 Agent |
| 建立索引 | index-docs.md | 目錄需要 index.md 索引時 | 任何 Agent |
| 下一步建議 | help.md | 不確定下一步該做什麼時 | 任何 Agent |
