# 步驟 2：路由對象

**進度：步驟 2 / 共 3 步**

## 目標

根據分派原則，選擇最適合處理的高管作為路由對象。

## 執行規則

- 📖 讀完整個步驟檔案再開始行動
- 🚫 不預讀後續步驟
- 🚫 不跳步

## 執行指令

### 1. 路由對照表

| 指令類型 | 路由對象 | 說明 |
|---------|---------|------|
| 財務相關 | CFO | 記帳、預算、消費分析、成本審計 |
| 投資相關 | CIO | 投資組合、市場研究、買賣建議 |
| 生活相關 | COO | 行程、飲食、出行、天氣 |
| 技術相關 | CTO | 產品開發、程式碼、架構設計 |
| Agent 管理 | CHRO | 能力評估、政策草擬、模型調整 |
| 安全/稽核 | CAO | 安全掃描、合規檢查、稽核議題 |

### 2. 綜合任務拆分

若任務涉及多個部門：

1. 拆分為獨立子任務。
2. 為每個子任務確定路由對象。
3. 標記任務間的依賴關係（若有）。

### 3. 確認路由

- 確認所選高管是否為最佳人選。
- 若不確定，向董事長確認。

## 完成標準

- [ ] 已確定路由對象
- [ ] 綜合任務已拆分（若適用）
- [ ] 路由選擇已確認
