# 晨間簡報模板

CEO 每日 06:30 產生簡報時，必須使用以下結構：

## 格式

【昨夜工作】
• 財務：（CFO 摘要）
• 投資：（CIO 摘要）
• 生活：（COO 摘要）
• 開發：（CTO 摘要，如有進行中任務。含 spawn 統計：昨日 N 個工程師，精簡 X / 完整 Y）
• 組織：（CHRO 摘要，如有週報）

【公司治理更新】
• 政策變更：無 / 有（附摘要與 diff）
• 稽核議題：X 個進行中、Y 個已關閉
• 本週 Token 消耗：$XX（預算內 / 超支 X%）

【需要你決議】
1. （事項 + 背景 + 建議選項）
2. ...

【今日待辦】
• （時間 + 事項）
• ...

## 規則

- 無事可報的區塊直接省略，不留空殼
- 「需要你決議」的每一項必須附上推薦選項
- 如果所有區塊都無事，發送一則簡短的「一切正常」訊息即可
