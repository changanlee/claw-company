# Quality Checklist

Required checks before completing a Dev Story. Used during Step 3 verification.

---

- [ ] All tasks completed and marked `[x]`
- [ ] All tests passing (zero failures)
- [ ] Test coverage meets agreed standard
- [ ] Lint has no errors
- [ ] Acceptance criteria confirmed item by item
- [ ] No hardcoded secrets or sensitive information
- [ ] Code style is consistent
- [ ] Necessary documentation updated
- [ ] Dev Agent Record has documented the development process
