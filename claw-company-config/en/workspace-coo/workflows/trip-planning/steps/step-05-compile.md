# Step 5: Compile

## Goal

Consolidate transport, accommodation, and daily itinerary from previous steps into a complete trip plan draft.

## Execution Rules

- 📖 Read the entire step file before taking action
- 🚫 Do not pre-read subsequent steps
- 🚫 Do not skip steps

## Instructions

### 1. Read Template

Read `{{INSTALL_DIR}}/workspace-coo/templates/trip-plan.md` and follow the template format.

### 2. Consolidate Content

Integrate per template structure:
- Basic info (destination, dates, duration, party size)
- Transport plan (recommended + alternatives)
- Accommodation plan (recommended + alternatives)
- Daily itinerary (with alternatives)
- Budget summary (transport + accommodation + meals + activities + misc)
- Notes (weather, visa, insurance, etc.)

### 3. Budget Summary

Compile all cost items:
- Transport costs (round trip)
- Accommodation costs (total)
- Meal budget (daily estimate x days)
- Activity costs (tickets, experiences, etc.)
- Contingency (10-15% of total)
- **Grand total**

### 4. Self-Review

- Confirm no date/time conflicts
- Confirm budget within limits
- Confirm all necessary information is complete
