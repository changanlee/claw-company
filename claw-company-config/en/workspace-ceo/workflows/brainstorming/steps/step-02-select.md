# Step 2: Select Technique

**Progress: Step 2 of 5**

## Objective

Read the brainstorming technique library and let the Chairman choose a thinking technique mode.

## Execution Rules

- Read the entire step file before acting
- Do not pre-read subsequent steps
- Do not skip or merge steps

## Instructions

### 1. Read Technique Library

Read `{{INSTALL_DIR}}/shared/brain-methods.csv`.

### 2. Offer Four Modes

Present choices to the Chairman:

> Before we begin brainstorming, how would you like to select a thinking technique?
>
> 1. **Manual** — Browse 9 categories and pick techniques that interest you
> 2. **AI Recommended** — I'll recommend 2-3 best-fit techniques based on the topic
> 3. **Random** — Randomly draw techniques for surprise inspiration
> 4. **Direct Discussion** — Skip techniques and go straight to Q&A discussion

#### Mode 1 — Manual

1. List the 9 categories with the number of techniques in each.
2. After the Chairman selects a category, list all techniques in that category.
3. Chairman picks 1-3 techniques.

#### Mode 2 — AI Recommended

Recommend 2-3 techniques based on the topic, with rationale.

#### Mode 3 — Random

Randomly draw 2 techniques, showing name and description. Re-draw available.

#### Mode 4 — Direct Discussion

Skip techniques and proceed to traditional Q&A discussion.

## Completion Criteria

- [ ] Technique library read
- [ ] Chairman has selected mode and technique (or direct discussion)
