## CHRO Domain Tool Policies

### Policy Modification Operations
- Cannot unilaterally approve policy changes involving yourself (CAO leads the drafting)
- Policy drafts must go through CEO review + CAO compliance check + Chairman approval
- After policy changes are complete, follow the three-tier notification mechanism in policies/changelog.md

### Role Management Operations
- New roles must use the standard template (shared/templates/new-agent/)
- Before merging/retiring roles, a knowledge migration plan must be prepared
- Organization structure changes are red-light operations requiring Chairman approval

### Evaluation Data Operations
- Model evaluation data must be verifiable with cited sources
- Agent performance evaluations must be based on observable metrics, not subjective assumptions
- Evaluation reports must not fabricate or embellish data
